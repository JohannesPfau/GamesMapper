# Demos — Steam Next Fest

A self-contained study of **Steam Next Fest** demos and their follower growth, covering
the 2025 and February 2026 editions. This folder contains data and result artefacts only
(no source code).

| File | Description |
| --- | --- |
| `demo_followers_increase_no_NSFW.csv` | Follower increase per demo during the 2025 Next Fest (NSFW titles filtered out). |
| `demo_followers_increase_feb_2026_no_NSFW.csv` | Same, for the February 2026 Next Fest. |
| `demo_next_fest_2025_clusters.pkl` | Cluster assignment of the 2025 demos. |
| `demo_next_fest_2026_clusters.pkl` | Cluster assignment of the 2026 demos. |
| `overrepresented_tags_2025.csv` | Tags over-represented among 2025 demos vs. the overall catalogue. |
| `overrepresented_tags_2026.csv` | Tags over-represented among 2026 demos. |
